# Autonomous-Navigation-on-Unitree-GO1

This project focuses on autonomous indoor navigation and localization for the Unitree Go1 quadruped robot. The work involved collecting IMU sensor data, implementing an Unscented Kalman Filter (UKF)-based state estimation, and testing robot movement in indoor environments.

**Project Objectives**
1. Receive and process IMU sensor data
2. Implement UKF for localization and state estimation
3. Analyze sensor data collected from the robot
4. Test autonomous movement and navigation behavior

**Technologies Used**
1. Python
2. ROS2
3. Unitree Go1
4. IMU Sensors
5. UKF (Unscented Kalman Filter)

**Results**
1. Successfully collected and processed real-time IMU data
2. Improved localization stability using UKF filtering
3. Tested indoor robot movement and navigation behavior
